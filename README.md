# To-Do List

A simple To-Do List application made with Python.

This project was created to practice working with:

* Functions
* Dictionaries
* Loops and conditions
* File handling
* Input validation

## Features

* Add new tasks
* View all tasks
* Mark tasks as completed
* Delete tasks
* Save tasks to a text file
* Prevent empty task names
* Automatically rename duplicate tasks

## How Tasks Are Stored

Tasks are stored in a dictionary.

```python
{
    "Learn Python": False,
    "Finish Homework": True
}
```

* `False` → Pending
* `True` → Completed

## Menu

```text
1. Add Task
2. View Tasks
3. Complete Task
4. Delete Task
5. Save Tasks
6. Exit
```
